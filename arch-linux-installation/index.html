Arch Linux Installation

Note : Arch is one of my favorite GNU/Linux distributions, however it is not quite easy to install. So I document some procedures for future references. This is not a complete guide.

  • Boot from Arch CD/DVD
  • Prepare the hard drives
fdisk -l  

cgdisk /dev/sda  
mkfs.vfat -F32 -n efi /dev/sda1 # -F## is not required on most firmwares; FAT12/16 usually work  
mkfs.ext4 -L "Arch Linux" /dev/sda2 # / mount point  
mkfs.ext4 -L "Home" /dev/sda2 # /home mount point  

mount /dev/sda2 /mnt  
mkdir -p /mnt/boot  
mount -o noatime /dev/sda1 /mnt/boot

If "Failed to mount '/dev/sda1': Invalid argument" 

ntfsfix /dev/sda1  

nano /etc/pacman.d/mirrorlist

Scroll down to your preferred mirror (the closer to your location the better), press Alt+6 to copy the line, then scroll back up and press Ctrl+U to paste that line at the top of the list.

  • Install the Arch Base System
pacstrap -i /mnt base base-devel grub-efi-x86_64  

genfstab -U -p /mnt >> /mnt/etc/fstab  
nano /mnt/etc/fstab  

arch-chroot /mnt /bin/bash  

nano /etc/locale.gen  
locale-gen  
echo LANG=en_US.UTF-8 > /etc/locale.conf  
export LANG=en_US.UTF-8  

ls /usr/share/zoneinfo  
ln -s /usr/share/zoneinfo/Asia/Ho_Chi_Minh /etc/localtime  
pacman -S ntp  
ntpd -qg  
hwclock --systohc  

echo kozyice > /etc/hostname

(Exchange 'kozyice' with the desired name for the system.)

  • Set Up Your Network
ip link

Wired Connection

systemctl enable dhcpcd@enp3s0.service

Wireless Connection

pacman -S wireless_tools wpa_supplicant wpa_actiond dialog  
wifi-menu  
systemctl enable net-auto-wireless.service
  • Configure Your Package Manager
nano /etc/pacman.conf

Scroll down to the "Repositories" section. The [core], [extra], and [community] repositories should already be uncommented, but if you're on a 64-bit machine, you should also enable the [multilib] repository, which lets you install both 64- and 32-bit programs. To do so, add the following lines to the bottom of the config file:

[multilib]
Include = /etc/pacman.d/mirrorlist

Refresh your repository list: 

pacman -Sy
  • Create an initial ramdisk environment
mkinitcpio -p linux
  • Create a User Account

First, set the root password: 

passwd

Create a user account (and a password) for yourself: 

useradd -m -g users -G wheel,storage,power -s /bin/bash hardy  
passwd hardy

(Of course, replace 'hardy' with your desired username.)

Install sudo: 

pacman -S sudo  
EDITOR=nano visudo

Scroll down until you find this line and uncomment it: 

%wheel ALL=(ALL) ALL

Install bash-completion: 

pacman -S bash-completion
  • Install Your Bootloader (for UEFI motherboards)
pacman -S gummiboot  
gummiboot --path=/boot install  

nano /boot/loader/entries/arch.conf  

title          Arch Linux  
linux          /vmlinuz-linux  
initrd         /initramfs-linux.img  
options        root=/dev/sdaX rw  

nano /boot/loader/loader.conf  

timeout       6  
default       arch

Lastly, exit from chroot, unmount your partitions, and reboot your computer:

exit  
umount /mnt/home  
umount /mnt  
reboot
  • Get Your Desktop Up and Running
sudo pacman -S xorg-server xorg-server-utils xfce4

Go ahead with XFCE.
